# 회의비 관리 시스템

백엔드 API를 기반으로 한 회의비 처리 웹 애플리케이션입니다.

## 기능

### 로그인 시스템
- 사용자명/비밀번호 기반 로그인
- 권한별 페이지 접근 제어 (일반 사용자/관리자)

### 일반 사용자 기능
- 회의 등록 (제목, 설명, 날짜, 시간, 예상 비용)
- 본인이 등록한 회의 목록 조회
- 회의 상태별 필터링
- 승인된 회의비 통계

### 관리자 기능
- 사용자 관리 (생성, 조회, 삭제)
- 회원가입 처리 (관리자가 직접 사용자 등록)
- 회의 상태 관리 (예정/진행중/완료/취소)
- 전체 회의비 통계 및 승인 관리

## 기술 스택
- React 19
- React Router DOM (라우팅)
- Styled Components (스타일링)
- Axios (HTTP 클라이언트)
- Context API (상태 관리)

## 실행 방법

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 백엔드 서버가 http://localhost:8080에서 실행 중이어야 합니다.

## API 연동
- 사용자 API: `/api/users`
- 회의 API: `/api/meetings`
- 회의 상태 업데이트: `/api/meetings/{id}/status`

## 로그인 정보
백엔드에 등록된 사용자의 userName으로 로그인 가능합니다.
관리자 권한은 role이 'ADMIN'인 사용자에게 부여됩니다.
